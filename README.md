# 인도형의 찬양 기타코드 대백과(by 전도사닷컴)

초보자와 찬양팀 기타 연주자를 위한 무료 통기타 코드 탐색기 MVP입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 포함된 MVP

- 루트와 코드 타입 버튼 선택
- 기본, 반주, 텐션, 얼터드, 분수코드 그룹
- 연주 상황 모드별 보이싱 정렬
- SVG 코드 다이어그램
- 손가락 번호, 바레, 루트음 표시
- 즐겨찾기와 최근 본 코드
- 왼손잡이 모드와 고급 운지 표시 설정

## 검증

```bash
npm run lint
npm run build
```
